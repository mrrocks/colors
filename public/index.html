This tool uses the OKLCH color format to generate a perceptually uniform full-spectrum color palette, ensuring that changes in color values correspond to equal changes in perceived luminosity and contrast. It supports a wider color gamut for accurate representation on modern displays and uses APAC to check contrast ratios. Additionally, it employs the CIEDE2000 formula to scientifically calculate color difference. Learn more about OKLCH's benefits [1], [2], [3], and APAC's advantages here.
